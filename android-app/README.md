# Дневник — Android App

Мобильное приложение для личного дневника с локальным хранением данных (SQLite на устройстве).

## Функционал

- **Записи дня** — что сделал, что не вышло, оценка дня 1-10
- **Задачи** — план на следующий день, управление статусами (выполнено/перенесено/отменено)
- **Статистика** — график настроения за 7/14/30 дней, процент выполнения задач
- **AI анализ** — общий анализ, психологический анализ, work-life баланс (через OpenRouter)
- **Уведомления** — утренние и вечерние напоминания
- **Экспорт** — выгрузка дневника в .txt файл

## Установка и запуск

```bash
cd android-app
npm install
npx expo start
```

Открой в Expo Go на Android (сканируй QR-код).

## Сборка APK

### Через EAS Build (облако, рекомендуется):

```bash
npm install -g eas-cli
eas login
eas build -p android --profile preview
```

APK будет доступен для скачивания через ссылку от EAS.

### Локальная сборка:

```bash
# Нужен Android Studio + JDK 17
npx expo run:android
```

## Настройка

1. Открой приложение → вкладка «Ещё»
2. Укажи OpenRouter API ключ для AI анализа (получи на openrouter.ai)
3. Настрой время уведомлений
4. Заполни профиль для персонализации AI

## Структура данных

Все данные хранятся локально в `diary.db` (SQLite) на устройстве.
Данные **не стираются** при обновлении приложения.

## Технологии

- React Native + Expo SDK 51
- expo-sqlite — локальное хранилище
- expo-notifications — уведомления
- react-native-chart-kit — графики
- OpenRouter API (Gemini 2.5 Flash) — AI анализ
