# Дневник — Android App

Мобильное приложение для личного дневника с локальным хранением данных, трекером задач, статистикой и AI-анализом.

## Функционал

### Записи дневника
- Пошаговое заполнение итога дня: что сделал, что не вышло, оценка дня 1–10, планы на завтра
- Просмотр всех прошлых записей, редактирование через календарь
- AI генерирует персональный совет на следующий день после сохранения записи

### Задачи
- Задачи на сегодня, завтра, просроченные и без даты
- Действия по долгому нажатию: выполнено, перенести, отменить, удалить
- Уведомление о просроченных задачах прямо на главном экране

### Статистика
- График настроения за 7 / 14 / 30 дней
- Процент выполнения задач, счётчики переносов и отмен

### AI анализ (требует OpenRouter API ключ)
- **Общий анализ (7 и 30 дней)** — темы, баланс сфер жизни, динамика настроения, паттерны
- **Психологический анализ** — незакрытые паттерны, разрыв план/реальность, что даёт и забирает энергию, 3 конкретные рекомендации
- **Work-life баланс** — что вытесняется, что работает, план на неделю
- Результат любого анализа можно отправить через стандартный share sheet

### Настройки
- Тема: авто / светлая / тёмная
- Утренние и вечерние push-уведомления с настраиваемым временем
- Профиль (пол, семейный статус, bio) — используется для персонализации AI
- OpenRouter API ключ

### Данные
- Экспорт дневника в `.txt` файл
- Импорт из ранее экспортированного файла (не перезаписывает существующие записи)
- Все данные хранятся локально в SQLite на устройстве

## Установка и запуск

```bash
cd android-app
npm install
npx expo start
```

Открой в Expo Go на Android (сканируй QR-код).

## Сборка APK

### Через EAS Build (облако, рекомендуется):

```bash
npm install -g eas-cli
eas login
eas build -p android --profile preview
```

APK будет доступен для скачивания по ссылке от EAS.

### Локальная сборка:

```bash
# Нужен Android Studio + JDK 17
npx expo run:android
```

## Настройка AI

1. Получи бесплатный ключ на [openrouter.ai](https://openrouter.ai)
2. Открой приложение → вкладка «Ещё» → OpenRouter API ключ
3. Заполни профиль для персонализации анализов

Используются модели **Gemini 2.5 Flash** (глубокие анализы) и **Gemini 2.0 Flash Lite** (быстрые советы).

## Технологии

- React Native + Expo SDK 52
- expo-sqlite — локальное хранилище
- expo-notifications — уведомления
- react-native-chart-kit — графики настроения
- OpenRouter API — AI анализ
