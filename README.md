# 📓 Diary Bot

Личный бот-дневник с AI-анализом, трекером планов и проактивными советами по work-life балансу.

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Создать файл .env
cp .env.example .env
# Вписать BOT_TOKEN и ANTHROPIC_API_KEY

# 3. Запустить
npm start
```

## Два пользователя с разным расписанием

После `/start` каждый пользователь настраивает своё время:

**Ты:**
```
/время_утро 09:00
/время_вечер 21:00
```

**Жена:**
```
/время_утро 10:00
/время_вечер 22:00
```

## Команды

| Команда | Описание |
|---|---|
| `/итог` | Вечерний итог дня |
| `/чекин` | Утренний чекин по планам |
| `/сегодня` | Запись за сегодня |
| `/неделя` | Последние 7 дней |
| `/настроение` | График оценок дня |
| `/планы` | Активные задачи |
| `/хвост` | Задачи с 2+ переносами |
| `/анализ 7` | Общий анализ за 7 дней |
| `/анализ 30` | Общий анализ за месяц |
| `/психо` | Психологический анализ |
| `/баланс` | Work-life советы |
| `/экспорт` | Выгрузка в .txt |

## Дневной ритм

```
Утро (09:00 / 10:00)  ──  Чекин по вчерашним планам
Вечер (21:00 / 22:00) ──  Итог дня + планы на завтра
                           + проактивный совет от AI
```

## Структура проекта

```
diary-bot/
├── index.js    — основной бот, все хендлеры
├── ai.js       — AI-анализ (Claude)
├── db.js       — SQLite база
├── .env        — токены (не коммитить!)
└── diary.db    — база данных (создаётся автоматически)
```

## Запуск в фоне на сервере

```bash
npm install -g pm2
pm2 start index.js --name diary-bot
pm2 save
pm2 startup
```
